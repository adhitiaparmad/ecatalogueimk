## eCatalogue IMK Project

eCatalogue IMK Project adalah sebuah aplikasi untuk mencari buku yang dibangun untuk gramedia dalam rangka tugas mata kuliah IMK di universitas paramadina.

Teknologi yang digunakan :

- Bahasa pemograman : PHP
- Database : mysql
- Framework : Laravel 5.4
